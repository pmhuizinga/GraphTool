This code will make a shorthand Ajax request for JSON-encoded data. This data contains a list of the values for your food select box. For this to work you will need to add an endpoint /get_food/ to your Flask views: food = { 'fruit': ['apple', 'banana', 'cherry'], 'vegetables': ['onion', 'cucumber'], 'meat': ['sausage', 'beef'], } @app.route('/get_food/') def get_food(foodkind): if foodkind not in food: return jsonify([]) else: return jsonify(food[foodkind])
